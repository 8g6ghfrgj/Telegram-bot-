# Telegram Bot - بوت تلجرام متكامل

بوت متكامل لإدارة الحسابات، الإعلانات، المجموعات، والردود التلقائية.

## المميزات:
- إدارة حسابات تلجرام متعددة
- نشر إعلانات تلقائية في المجموعات
- الرد التلقائي على الرسائل
- إدارة المشرفين
- ردود نصية وصور في المجموعات

## التثبيت:
1. انسخ المشروع
2. اضبط BOT_TOKEN
3. شغل البوت

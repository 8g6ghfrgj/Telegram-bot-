# Telegram Bot - متعدد المهام

بوت تيليجرام متكامل لإدارة الحسابات والنشر التلقائي.

## المميزات:
- إدارة حسابات متعددة
- نشر تلقائي في المجموعات
- ردود تلقائية في الخاص والمجموعات
- إدارة مشرفين متعددة

## التنصيب على Render.com

1. أضف هذا المستودع إلى Render
2. أضف متغير البيئة BOT_TOKEN
3. شغّل البوت

## الأوامر:
- `/start`: بدء البوت
